# docker-lemp-stack

## Stack
- [alpine] nginx v1.15
- [alpine] php-fpm(php7.2)
- [alpine] memcached v1.5
- [debian] mysql v5.7

## Get started

```
$ git clone https://github.com/oshou/docker-lemp-stack.git
$ cd docker-lemp-stack
$ export SRC_DIR=/path/to/your_application_source_directory
$ docker-compose up -d
```
